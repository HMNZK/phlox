Phlox
EN 日本語

プライバシーポリシー

施行日: 2026-07-14

概要

Phlox は、複数の AI コーディングエージェントを1つのウィンドウから操作できる macOS アプリです。 Phlox は解析（アナリティクス）も個人データも一切収集しません。 アプリはお使いのマシン上で動作し、利用情報を当方へ送信することはありません。

収集しない情報

Phlox は以下を一切収集・送信しません:

  • 氏名・メールアドレスその他の個人を特定できる情報
  • いかなる利用解析・テレメトリも行いません
  • ファイルパス・セッション名・ワークスペース名
  • AI エージェントへ送るプロンプトや、その応答
  • コードやファイルの内容

第三者サービス

  • GitHub — アプリの配布・リリースをホストします。ダウンロードや更新確認は GitHub の集計ダウンロード数に反映されることがありますが、個人を特定するものではありません。
  • Sparkle — オープンソースの macOS 自動更新フレームワーク。新しい版の有無を確認するため更新フィードへ通信しますが、個人データは送信しません。
  • AI エージェント CLI — Phlox は、利用者自身がインストール・設定する第三者の AI コーディングツール（Claude Code・Codex・Cursor 等）を操作します。それらを通じて送信される内容は、各ツールのプライバシーポリシーに従って扱われ、Phlox の対象外です。
