Privacy Policy

Effective date: 2026-07-14

Overview

Phlox is a macOS application that lets you orchestrate multiple AI coding agents from a single window. Phlox collects no analytics and no personal data. The app runs on your own machine and does not send usage information to us.

Information not collected

Phlox does not collect or transmit any of the following:

  • Name, email address, or any other personally identifiable information
  • Usage analytics or telemetry of any kind
  • File paths, session names, or workspace names
  • Prompts sent to AI agents or the responses received from them
  • Code or file contents

Third-party services

  • GitHub — hosts the app's distribution and releases. Downloading the app or checking for updates may be reflected in aggregate download counts provided by GitHub; these do not identify individual users.
  • Sparkle — the open-source macOS update framework. It makes network requests to the update feed to check for new versions; no personal data is sent.
  • AI agent CLIs — Phlox drives third-party AI coding tools (such as Claude Code, Codex, and Cursor) that you install and configure yourself. Anything you send through those tools is handled under their own privacy policies, not Phlox's.

Changes to this policy

If this policy changes materially, the updated version will be posted at this URL with a new effective date.

Contact

Questions about this policy can be raised on the project's GitHub issue tracker: github.com/HMNZK/phlox/issues
